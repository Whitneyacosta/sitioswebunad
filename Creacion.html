Creación de repositorio remoto en Ubuntu

Instalación de git

Lo primero que debemos hacer es crear una cuenta en https://github.com/ y como segundo abrir el terminal de nuestro sistema operativo ubuntu, y seguidamente ejecutar el comando:

Sudo apt install git-core
Una vez se hallan descargado los paquetes de la instalación ejecutan el siguiente comando para verificar su versión y que se halla instalado:
git --version

Configuración de git en Ubuntu

Despues de la instalación configuraremos la cuenta con los comandos:

git config --global user.name "Whitney"
git config --global user.email "wlacostarobelto@misena.edu.co"

Clonación de repositorio

Una vez creada la cuenta y un repositorio en https://github.com/ haremos la clonación en el equipo con sistema operativo Ubuntu.
Ejecutamos el comando:

git clone:(Más la URL de nuestro repositorio)//github.com/whitneyacosta/whitney_pruebas

Instalación de Gui

Una vez hecha la clonación del repositorio procedemos con la instalación de Gui, el cual nos permitira hacer cargues de commit es decir el cargue de los documentos de una página web o los documentos que necesitemos bajo cualquier lenguaje de programación.

git gui
